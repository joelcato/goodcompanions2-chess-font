## Key technical notes

### make_gc_pfb.py
Requires FontForge (`fontforge -script make_gc_pfb.py`). The `import fontforge`
line has `# type: ignore` to suppress Pylance false positives (FontForge ships
its own Python C extension, no pip stubs exist).

### Why two TTF files?
- `goodcompanions.ttf` ‚Äî original, unmodified; chess glyphs at Latin codepoints, 172 total glyphs
- `chess-goodcompanions2.ttf` ‚Äî prepared TTF; 26 chess glyphs only, renamed to `chess-board.enc`
  names (`k`, `K`, `q`, ‚Ä¶) with unicode values set to the enc slot codepoints.
  Any TTF‚Üípfb converter applied to this file gives a correct pfb without needing the script.

### Font map registration
The two map entries in `chess-goodcompanions2.map`:
1. Raw entry (font without re-encoding) ‚Äî needed so pdftex can find the pfb
2. LSB entry with `ChessBoardEncoding ReEncodeFont` ‚Äî this is what `chessfss` uses at runtime

### TFM generation command
```bash
afm2tfm chess-goodcompanions2-board-fig-raw.afm \
  -T $(kpsewhich chess-board.enc) \
  chess-goodcompanions2-lsb
```
